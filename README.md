# ☕ Maratona Java

Repositório criado para registrar toda minha jornada estudando Java do zero até tópicos mais avançados.

A ideia não é apenas "acompanhar um curso", mas construir meu próprio material de consulta ao longo do tempo.  
Quero transformar esse projeto em uma espécie de guia pessoal onde eu consiga revisitar conceitos rapidamente quando esquecer algo no futuro.

---

# 📚 Conteúdos

Este repositório cobre desde fundamentos básicos até conceitos mais avançados da linguagem:

- Variáveis e tipos primitivos
- Operadores
- Estruturas condicionais
- Laços de repetição
- Arrays
- Métodos
- Orientação a Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Classes abstratas
- Interfaces
- Exceptions
- Collections
- Generics
- Streams
- Lambdas
- Modificadores
- Boas práticas
- E muito mais...

# 🎯 Objetivo

Mais do que terminar uma "maratona", o objetivo é:

- consolidar fundamentos
- praticar lógica
- reforçar conceitos importantes
- criar exemplos próprios
- manter um repositório de referência feito por mim mesmo
